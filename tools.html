chm 工具

微软HTML帮助集，即编译的HTML帮助文件（英语：Microsoft Compiled HTML Help, CHM），是微软继承早先的WinHelp发展的一种文件格式，用来提供在线帮助，是一种应用较广泛的文件格式。因为CHM文件如一本书一样，可以提供内容目录、索引和搜索等功能，所以也常被用来制作电子书。
近期有合并chm的需求，但从网上找了很多的方法，都不行。过程中接触到多个chm软件，一下以各个软件的。

  • HTML Help workshop
    这个软件是微软推出的制作chm的软件, 查看帮助文档后，可以制作出chm文档，但在合并文件后总是报错（HHC5013: Error: URL reference in the TOC cannot be resolved:***)，错误无法解决。
  • ABC Amber CHM Merger
    这个工具很好用，但是有以下的缺陷。
  1. 合并的chm不支持中文语言，导致合并后主题乱码
  2. 合并后没有所有窗口，原来的搜索窗口丢失
    注：在合并时需要指定hhc.exe的位置
  • RoboHTML
    找到了下面的指导文档，但时无法安装上面的工具，哎哎哎！心累。。。
    http://www.hypertexas.com/RHTutor/merger.htm
